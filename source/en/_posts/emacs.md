[](#Space-Series)Space Series
Space key space starts with very many shortcut keys, viewing method, press Space key, instantly pops up the first layer of shortcut keys bound with space. After that, there's also a second layer of shortcut keys starting with keywords.

[](#f-Series)+f Series
Is file-related. Files that need to be edited directly using sudo commands within emacs can use space + f + E to start.

[](#p-Series-Is-Project-Related,)+p Series Is Project-Related,
Like +p enter, find space+p+t is viewing root directory's nerdtree file directory management.

[](#b-buffer-Series)+b buffer Series
Directly space+b+b recent edited files' buffer一览无余.

[](#g-git-Series)+g git Series
Various shortcut keys for git commands. On rainy days, magit and emacs go better together.

[](#h-Help)+h Help
space+h is the prefix shortcut key for various emacs shortcut keys

[](#shell-Command-Shortcut-Key)+! shell Command Shortcut Key
Directly execute shell commands. So cool~

[](#n-Number-Window-Switch)+n Number Window Switch
Is switching between window numbers 1-n.
