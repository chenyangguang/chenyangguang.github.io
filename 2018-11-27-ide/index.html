Emacs as golang IDE

2018.11.27 Gitvim  热度 ℃

IDE

据我观察，很多程序员们喜欢 IDE 的原因在于工程化全局查询，函数定义, 复制粘贴， 其他的模块化插入代码块，是没有太多吸引力的。

作为 Go 开发 IDE

emacs as golang ide 这里有基本的配置。
godef 安装一下可以解决函数定义的出处查找问题。没事就来一下: C-c C-j, 立马追到底。包括 go/src/ 目录也能追杀到底。
